# Bundesländer Quiz

This project is a quiz application to help people learn about the federal states of Germany (Bundesländer). The app provides engaging questions and tracks the user's progress.

## Features
- Interactive quiz format
- Tracks user progress
- Covers all 16 German federal states

## Getting Started
To get started, clone this repo and open the application to start the quiz!
